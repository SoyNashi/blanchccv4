# Instrucciones para el Portfolio de Nil Blanch

- **Estética**: Premium, minimalista, fondo oscuro.
- **Tecnologías**: Next.js 15, Tailwind CSS, Framer Motion.
- **Estructura**:
  - Hero: Mensajes contundentes rotativos.
  - Wall: Proyectos principales (Blau360, Bastet).
  - Editorial: Capacidades/Servicios.
  - Certificaciones: Marquee doble infinito.
  - Blog: Sistema preparado para DB/Markdown con categorías.
- **Animaciones**: React Bits, suaves, elegantes.
- **SEO**: Configurado en layout y páginas de blog.

## Base de Datos
Para conectar la base de datos de Ionos, configurar `DATABASE_URL` en las variables de entorno.
El sistema de blog está preparado para leer tanto de JSON (fallback) como de DB para likes/comentarios.
